Usage

使用方式 — <i class="icon icon-{name}"></i>，尺寸用 font-size，顏色繼承 currentColor。

HTML

<i class="icon icon-search" aria-hidden="true"></i>
<i class="icon icon-heart" style="font-size:20px;color:var(--cl-red-500)"></i>

並文字使用

<button class="gl_btn gl_btn-md gl_btn-success">
  <i class="icon icon-search"></i>
  <span>搜尋職缺</span>
</button>

規則

  • Inline size 推薦 14 / 16 / 20 px；特外用途（empty state、feature）才用 24–48。
  • 不設顏色時繼承父元素 color；要收弱即設 color: var(--fg-3)。
  • 純裝飾性 icon 加 aria-hidden="true"，有語意的加 aria-label。
  • 與文字同行時，icon 與 text 之間用 gap: 8px 或 margin-right: 4–8px。
  • Icon-only 按鈕統一用 .gl_btn-icon，且必須有 aria-label。
