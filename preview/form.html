Form · Input · Select · Textarea

.gl_form-group 包住每個欄位。高度 48px、圓角 8px。

INPUT

SELECT · TEXTAREA

PROMPT / HINT

以新台幣計算
同職位中位數為 NT$ 52,800

VALIDATION

Email 格式正確
缺少 @ 與網域
Usage .gl_form-group · .gl_form-control 
<!-- basic -->
<div class="gl_form-group">
  <input class="gl_form-control" type="text" placeholder="職稱" />
</div>

<!-- with leading icon -->
<div class="gl_form-group gl_icon-input-wrap">
  <i class="icon icon-search"></i>
  <input class="gl_form-control" type="text" />
</div>

<!-- hint -->
<div class="gl_form-group">
  <input class="gl_form-control" />
  <div class="form-prompt-text is-info">中位數 NT$ 52,800</div>
</div>

<!-- validation -->
<div class="gl_form-group is-invalid">
  <input class="gl_form-control" />
  <div class="form-feedback">缺少 @ 與網域</div>
</div>

Classes

  • .gl_form-group：每個欄位的包裝。修飾用 .is-valid / .is-invalid / .gl_icon-input-wrap
  • .gl_form-control：<input>、<select>、<textarea>都套同一個 class
  • .form-prompt-text（hint）可加 .is-info / .is-warning
  • .form-feedback（validation）顏色由父層 .is-valid / .is-invalid 決定

規則

  • Hint 與 feedback 都在 input 下方
  • 高度 48px、radius 8px、tinted 底，focus 時翻白
  • Disabled 狀態直接在 <input> 加 disabled

Checkbox · Radio · Toggle

20 × 20 方框，與標籤間距 12px。主題變體 -success / -info / -danger / -basic。

CHECKBOX

RADIO

TOGGLE · sizes × colors

↑ lg · md · sm · xs
↑ basic · info · warning · danger · off
Usage .gl_checkbox-layout · .gl_radio-layout · .gl_toggle 
<!-- checkbox -->
<div class="gl_checkbox-layout gl_checkbox-success">
  <label><input type="checkbox" checked /><span>訂閱職缺摘要</span></label>
</div>

<!-- radio -->
<div class="gl_radio-layout gl_radio-success">
  <label><input type="radio" name="job" /><span>正職</span></label>
</div>

<!-- toggle -->
<label class="gl_toggle gl_toggle-md gl_toggle-success">
  <input type="checkbox" class="toggle-checkbox" />
  <span class="knobs"></span><span class="layer"></span>
</label>

Classes

  • Checkbox / Radio：.gl_[checkbox|radio]-layout + 顏色 -success / -info / -danger / -basic
  • Toggle：.gl_toggle + size -lg / -md / -sm / -xs + 顏色修飾
  • Toggle 內部必須有 <input>、.knobs、.layer

規則

  • 單選 Radio、多選 Checkbox、即時生效 Toggle
  • 破壞性選項（同意刪除）用 -danger
