陰影與邊框 Shadow & Border

三組工具：Border（outline 描邊，不佔空間）、Shadow（六階陰影）、Ring（= Shadow + Border 組合，無獨立 class）。直接把 class 疊加即可，例如 class="gl_shadow-md gl_border-outer"。

Border

.gl_border-* 用 outline 實作，不影響 box model，可疊在 shadow 上。

gl_border-outer
.gl_border-outer {
  outline: 1px solid $cl-basic-alpha-100;
}
gl_border-inner
.gl_border-inner {
  outline: 1px solid $cl-basic-alpha-100;
  outline-offset: -1px;

  @media (max-width: $bp-md) {
    outline: 1px solid $cl-basic-alpha-50;
    outline-offset: 0;
  }
}

Shadow

.gl_shadow-* 六階。卡片預設 sm、hover 升 md、modal 用 xl。

Ring

Ring = Shadow + Border 兩個 class 疊加，沒有獨立的 ring class / 變數。outer 用 .gl_border-outer，inner 用 .gl_border-inner。
