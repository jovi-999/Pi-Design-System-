Pagination

.iw_pagination-outer-v3 > .iw_pagination-wrap > prev 箭頭 + .gl_pagination-group + next 箭頭。當前頁 <li> 加 .iw_active；prev/next 為 .gl_page-link，末/首頁時加 .is-disabled。

Usage .iw_pagination-outer-v3 
<div class="iw_pagination-outer-v3">
  <div class="iw_pagination-wrap">
    <a class="gl_page-link is-disabled" href="?page=1" data-page="1" aria-label="上一頁"><i class="icon icon-arrow-left"></i></a>
    <ul class="gl_pagination-group">
      <li class="page iw_active"><a class="gl_page-link" href="?page=1" data-page="1">1</a></li>
      <li class="page"><a class="gl_page-link" href="?page=2" data-page="2">2</a></li>
      <li class="page"><a class="gl_page-link" href="?page=3" data-page="3">3</a></li>
      <li class="page"><a class="gl_page-link" href="?page=4" data-page="4">4</a></li>
      <li class="page"><a class="gl_page-link" href="?page=5" data-page="5">5</a></li>
    </ul>
    <a class="gl_page-link" href="?page=2" data-page="2" aria-label="下一頁"><i class="icon icon-arrow-right"></i></a>
  </div>
</div>

結構

  • 三層：.iw_pagination-outer-v3（最外層）> .iw_pagination-wrap（列，箭頭推兩端）> prev 箭頭 + .gl_pagination-group（數字組）+ next 箭頭
  • prev / next 箭頭是 wrap 的直接子 .gl_page-link，不放進數字 <ul>
  • 每個 .gl_page-link 帶 data-page；箭頭帶 aria-label

Sizes

  • cell 40px；≤768px（tablet 以下）自動縮為 32px
  • 箭頭用 icon-arrow-left / icon-arrow-right
  • 當前頁深色實心（basic），非當前頁與箭頭白底

規則

  • 當前頁 <li> 加 .iw_active
  • 首頁時 prev、末頁時 next 加 .is-disabled
  • < 2 頁則不顯示 Pagination
