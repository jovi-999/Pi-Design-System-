Loading · Spinner

Loading · Spinner

.gl_loading 讓 icon-_loading 以 2.5s 線性無限旋轉。

Usage .gl_loading 
<!-- standalone -->
<span class="gl_loading" style="font-size: 24px">
  <i class="icon icon-_loading"></i>
</span>

<!-- inside button -->
<button class="gl_btn gl_btn-md gl_btn-success" disabled>
  <span class="gl_loading"><i class="icon icon-_loading"></i></span>
  <span>送出中…</span>
</button>

規則

  • 尺寸用 font-size、顏色繼承 currentColor
  • 按鈕 loading 時必須加 disabled，避免重複送出
  • 頁面全屏 loading 置中，下方可選配標題文字
