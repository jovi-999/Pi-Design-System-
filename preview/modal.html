對話框 Modal

Dialog Modal

以原生 <dialog> 作為 modal 元素，外觀沿用 GSAP Modal 的樣式 class（gl_gs-modal）。開啟與關閉行為交給原生 API：showModal() / close()，::backdrop 遮罩、focus trap、Esc 關閉、:modal 置中皆由瀏覽器提供。GSAP 只處理進場與退場過渡。

// 開啟：原生 API 控制狀態，GSAP 只處理過渡
openBtn.addEventListener('click', () => {
  dlg.showModal();
  gsap.fromTo(dlg, { autoAlpha: 0, y: 32 }, {
    autoAlpha: 1,
    y: 0,
    duration: 0.35,
    ease: 'power2.out',
    clearProps: 'opacity,transform,visibility',
  });
});

// 關閉：GSAP 完成退場後，仍由原生 API 關閉
function animateDialogOut() {
  gsap.to(dlg, {
    autoAlpha: 0,
    y: 32,
    duration: 0.25,
    ease: 'power2.in',
    onComplete: () => {
      dlg.close();
      gsap.set(dlg, { clearProps: 'opacity,transform,visibility' });
    },
  });
}

closeBtn.addEventListener('click', animateDialogOut);

為何用 <dialog>

  • 原生 showModal() 自帶 ::backdrop 遮罩、focus trap、Esc 關閉，不需自造 overlay 層
  • 開關由原生 showModal() / close() 控制；GSAP 只處理進場與退場過渡
  • 置中由 :modal UA 樣式提供；外觀完全沿用 gl_gs-modal 樣式 class，只是容器換成 <dialog>

原生行為

  • 開啟：dlg.showModal()；關閉：dlg.close()（X 與 footer 按鈕皆同）
  • focus trap 由瀏覽器處理；Esc 的原生 cancel 事件只延後關閉，讓退場動畫先完成
  • 暫不用 invoker command（command / commandfor），以向下相容舊瀏覽器

GSAP Modal

三段式 header（左操作 / 中 title 置中 / 右關閉）+ body + footer。Header 與按鈕樣式沿用 Dialog Modal：三欄 grid 搭配 gl_btn-icon、gl_btn-sm、gl_btn-gray-basic。命名空間 gl_gs-modal-*（gs = GSAP），與 reveal modal 的 gl_modal-* 完全獨立。動畫由 tools/GsapModal.js + GSAP timeline 驅動。

結構

  • 兩層必備：gl-v2_modal-overlay（遮罩）+ gl_gs-modal（本體），缺一不可
  • 三段式 header：左右各 56px 操作區 + 中間 1fr，讓 title 固定置中
  • header 按鈕統一用 gl_btn gl_btn-icon gl_btn-sm gl_btn-gray-basic，40px 按鈕加 8px margin
  • 含 badge / 副標時加 is-badge；無左側操作時以 56px spacer 保持對稱

命名規範

  • Overlay id {name}ModalOverlay、Modal id {name}Modal（camelCase）
  • 容器 class iw_{name}-modal gl_gs-modal（kebab-case）
  • JS hook：iw_js-toggle-{name}-modal-btn / iw_js-close-{name}-modal-btn（連字符，禁底線）
  • gl_gs-modal-* 為 GSAP 專用，與 reveal 的 gl_modal-* 完全獨立

JS 串接（tools/GsapModal.js）

  • 純開關用 GsapModal.bindAll([...]) 批次綁定
  • 開啟前需 fetch / 初始化 → new GsapModal({...}) 單一綁定，自己綁 click 後呼叫 .open()
  • 預設動畫：overlay 0.3s fade、modal 0.3s back.out(1.7) 彈出

Modal Legacy

.gl_modal —— 寬 352px、圓角 24px，深色模糊遮罩上的白色面板。Footer 支援水平 + 垂直按鈕排列。Legacy：新版請優先用 Dialog Modal / GSAP Modal。
