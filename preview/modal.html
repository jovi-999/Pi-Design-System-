對話框 Modal

Dialog Modal

以原生 <dialog> 作為 modal 元素，外觀沿用 GSAP Modal 的樣式 class（gl_gs-modal）。行為全交給原生 API：showModal() / close()，::backdrop 遮罩、focus trap、Esc 關閉、:modal 置中皆由瀏覽器提供，不依賴 GSAP。

檢舉這則內容
檢舉這則內容
Usage <dialog> native 
{{-- gl_gs-modal 直接掛在 <dialog>，不再包一層 div --}}
<button class="iw_js-open-gs-dialog-btn">開啟</button>

<dialog id="gsDialog" class="gs-dialog gl_gs-modal gl_shadow-xxl gl_border-outer">
  {{-- header / content / footer 同 gl_gs-modal --}}
  <button class="iw_js-close-gs-dialog-btn">送出</button>
</dialog>

// 開關：呼叫原生 API
openBtn.addEventListener('click', () => dlg.showModal());
closeBtn.addEventListener('click', () => dlg.close());

為何用 <dialog>

  • 原生 showModal() 自帶 ::backdrop 遮罩、focus trap、Esc 關閉，不需自造 overlay 層
  • 不依賴 GSAP；開關用少量 JS 呼叫原生 showModal() / close()
  • 置中由 :modal UA 樣式提供；外觀完全沿用 gl_gs-modal 樣式 class，只是容器換成 <dialog>

原生行為

  • 開啟：dlg.showModal()；關閉：dlg.close()（X 與 footer 按鈕皆同）
  • Esc、focus trap 皆為瀏覽器原生，無需額外處理
  • 暫不用 invoker command（command / commandfor），以向下相容舊瀏覽器

GSAP Modal

三段式 header（左 spacer / 中 title 置中 / 右毛玻璃 close）+ body + footer。命名空間 gl_gs-modal-*（gs = GSAP），與 reveal modal 的 gl_modal-* 完全獨立。動畫由 tools/GsapModal.js + GSAP timeline 驅動。

檢舉這則內容
我來回答 + 10 積分
Usage .gl_gs-modal（GSAP） 
<div id="reportModalOverlay" class="gl-v2_modal-overlay">
  <div id="reportModal" class="iw_report-modal gl_gs-modal">

    {{-- Header：左 spacer + title 置中 + 右 close --}}
    <div class="gl_gs-modal-header">
      <div class="gl_gs-modal-header-spacer"></div>
      <div class="gl_gs-modal-header-title fz-title-md fz-tit">標題</div>
      <button type="button" class="gl_gs-modal-header-close iw_js-close-report-modal-btn">
        <i class="icon icon-cross"></i>
      </button>
    </div>

    {{-- Body --}}
    <div class="gl_gs-modal-content iw_modal-body"> … </div>

    {{-- Footer --}}
    <div class="gl_gs-modal-footer"> … </div>

  </div>
</div>

結構

  • 兩層必備：gl-v2_modal-overlay（遮罩）+ gl_gs-modal（本體），缺一不可
  • 三段式 header：spacer 56px + title flex-1 + close 圓形 40px，spacer 與 close 對稱讓 title 置中
  • header title 是 flex-direction: column 容器，可放 badge / 副標
  • 無 navigation 的 modal（如 unlock）用頁面專屬 header class，close 改 absolute 定位

命名規範

  • Overlay id {name}ModalOverlay、Modal id {name}Modal（camelCase）
  • 容器 class iw_{name}-modal gl_gs-modal（kebab-case）
  • JS hook：iw_js-toggle-{name}-modal-btn / iw_js-close-{name}-modal-btn（連字符，禁底線）
  • gl_gs-modal-* 為 GSAP 專用，與 reveal 的 gl_modal-* 完全獨立

JS 串接（tools/GsapModal.js）

  • 純開關用 GsapModal.bindAll([...]) 批次綁定
  • 開啟前需 fetch / 初始化 → new GsapModal({...}) 單一綁定，自己綁 click 後呼叫 .open()
  • 預設動畫：overlay 0.3s fade、modal 0.3s back.out(1.7) 彈出

Modal Legacy

.gl_modal —— 寬 352px、圓角 24px，深色模糊遮罩上的白色面板。Footer 支援水平 + 垂直按鈕排列。Legacy：新版請優先用 Dialog Modal / GSAP Modal。
