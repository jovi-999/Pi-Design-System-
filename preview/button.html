Usage .gl_btn 
<!-- size × variant × color -->
<button class="gl_btn gl_btn-md gl_btn-success">搜尋職缺</button>

<!-- with icon -->
<button class="gl_btn gl_btn-md gl_btn-success">
  <i class="icon icon-search"></i>
  <span>搜尋</span>
</button>

<!-- icon only -->
<button class="gl_btn gl_btn-md gl_btn-icon gl_btn-gray-basic" aria-label="更多">
  <i class="icon icon-more-horiz"></i>
</button>

Classes

  • gl_btn + size (xl·lg·md·sm·xs) + variant (filled·gray·outline·ghost) + color (basic·info·success·warning·danger·orange·purple)
  • Icon-only 需加 .gl_btn-icon 與 aria-label

規則

  • 主 CTA 用 gl_btn-success；每屏會1個主按鈕
  • 破壞性動作用 gl_btn-danger，與 ghost 取消右部對置

Disabled

任一變體加 disabled 屬性或 .gl_disabled，透明度降為 0.32、游標 not-allowed。
