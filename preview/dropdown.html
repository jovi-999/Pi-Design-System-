Dropdown menu

.gl_dropdown-item + 尺寸。用於浮出的 popover 選單內。

MD

SM

Usage .gl_dropdown-item 
<div class="gl_dropdown"> <!-- popover container -->
  <button class="gl_dropdown-item gl_dropdown-item-md">
    <i class="icon icon-bookmark"></i>收藏這則職缺
  </button>
  <button class="gl_dropdown-item gl_dropdown-item-md" style="color: var(--cl-red-600);">
    <i class="icon icon-delete"></i>刪除
  </button>
</div>

Sizes

  • -md（預設）、-sm（緊湊，用於 toolbar menu）

規則

  • Icon 必置左邊，與字間距 8px
  • 破壞性 item 直接套 color: var(--cl-red-600)
  • Menu 項目最多 7 個；超過改用分組或 Select
